# Blinkit Sales Dashboard

## 📖 Project Overview

This project presents an interactive Power BI dashboard that analyzes Blinkit's sales performance, customer purchasing patterns, product categories, and key business metrics. The dashboard provides valuable insights to support data-driven business decisions.

---

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Power Query
- DAX

---

## 📂 Dataset

The dashboard is built using Blinkit sales data containing information about products, sales, outlet types, customer ratings, and item categories.

---

## 📊 Dashboard Features

- Total Sales Analysis
- Average Sales
- Number of Items Sold
- Average Customer Rating
- Sales by Item Type
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Outlet Location
- Fat Content Analysis

---

## 📈 Key Insights

- Identified top-performing outlet types.
- Compared sales across different outlet locations.
- Analyzed customer preferences by item category.
- Evaluated average ratings and sales performance.

1. Download the `.pbix` file.
